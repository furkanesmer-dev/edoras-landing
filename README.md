# Edoras Akademi Landing Page

Bu repo, **edorasakademi.com** ana sayfa (landing page) projesini içerir.

⚠️ Not: Bu proje, kocluk platformundan tamamen bağımsızdır.
Backend, API veya üyelik sistemi bu repo kapsamında değildir.

---

## Amaç

* Modern ve premium bir görünüm oluşturmak
* Mobil uyumluluğu iyileştirmek
* SEO performansını artırmak
* Sayfa hızını optimize etmek
* Dönüşüm (conversion) oranını artırmak

---

## Tasarım Yaklaşımı

* Beyaz / aydınlık tema
